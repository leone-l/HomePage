# 个人主页 - 玻璃态设计

这是一个现代化的玻璃态（Glassmorphism）风格个人主页，基于提供的设计图片实现。
参考https://github.com/imsyy/home

## ✨ 特性

- 🎨 **玻璃态设计** - 半透明毛玻璃效果，现代感十足
- ⏰ **实时时钟** - 显示当前日期和时间
- 📱 **响应式布局** - 完美适配各种屏幕尺寸
- 🔗 **社交链接** - 集成 GitHub、B站、QQ 等社交平台
- 🌐 **网站导航** - 博客、网盘、音乐等常用网站入口
- 💬 **每日名言** - 展示励志名言

## 📁 文件结构

```
/workspace/
├── index.html              # 主页面
├── styles/
│   └── style.css          # 样式文件
├── scripts/
│   └── main.js            # JavaScript 交互逻辑
├── assets/                # 资源文件夹
│   └── images/
│       └── hero-bg.jpg    # 背景图片（请替换为你自己的）
└── README.md              # 说明文档
```

## 🎯 使用说明

### 1. 替换背景图片

请将你喜欢的背景图片命名为 `hero-bg.jpg` 并放置在 `assets/images/` 目录下。

### 2. 自定义内容

- **修改个人信息**：编辑 `index.html` 中的文字内容
- **更换社交链接**：修改 `<a>` 标签的 `href` 属性
- **调整网站链接**：更新网站卡片的链接地址
- **修改天气信息**：在 `index.html` 中修改天气文本

### 3. 本地预览

直接在浏览器中打开 `index.html` 即可预览效果。

## 🎨 设计特点

### 颜色方案
- 主背景：风景图片 + 半透明遮罩
- 卡片背景：`rgba(255, 255, 255, 0.15)` 半透明白色
- 文字颜色：白色，带透明度调整

### 玻璃态效果
- `backdrop-filter: blur(20px)` - 背景模糊
- 半透明边框和阴影
- 柔和的渐变效果

### 字体
- 中文：Noto Sans SC
- 特殊字体：Pacifico（手写风格）
- 时间显示：Courier New（等宽字体）

## 📱 响应式断点

- **桌面端**：> 1200px - 双列布局
- **平板**：768px - 1200px - 单列布局
- **移动端**：< 768px - 紧凑布局

## 🔧 浏览器支持

- Chrome/Edge（推荐）
- Firefox
- Safari
- 其他支持 `backdrop-filter` 的现代浏览器

## 📝 注意事项

1. 请确保有网络连接，以便加载 Google Fonts 字体
2. 背景图片可以根据你的喜好替换
3. 网站链接需要根据实际情况修改
4. 天气信息为静态文本，如需动态数据请接入天气 API
